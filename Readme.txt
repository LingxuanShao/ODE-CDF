# Read me

The data underlying this article are available at https://github.com/LingxuanShao/ODE-CDF.

(1) The files "Table x Panel y.R" contain the codes used to produce the simulation results in Section 5 in the main article.

(2) The files "Table Sx Panel y.R" contain the codes used to produce the simulation results in Section S2 in the Supplementary Material.

(3) The files "Table 4 Panel 1.R" and "Table 4 Panel 2.R" comtain the codes utilized for analyzing the COVID-19 data in terms of parameter estimation and curve recovery in Section 6.

(4) The file "data.xlsx" provides Covid data, with the headline presenting the province name, number of cured cases, number of confirmed cases, and the number of deaths.
